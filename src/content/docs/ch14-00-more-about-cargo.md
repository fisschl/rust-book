  label: 概述

到目前为止，我们只使用了 Cargo 最基本的特性来构建、运行和测试我们的代码，但它能做的还有很多。在本章中，我们将讨论一些其他更高级的特性，向你展示如何执行以下操作：

- 通过发布配置自定义你的构建。
- 在 [crates.io](https://crates.io/) 上发布库。
- 使用工作空间组织大型项目。
- 从 [crates.io](https://crates.io/) 安装二进制文件。
- 使用自定义命令扩展 Cargo。

Cargo 能做的比我们在本章中涵盖的功能还要多，因此要了解其所有特性的完整解释，请参阅[其文档](https://doc.rust-lang.org/cargo/)。
