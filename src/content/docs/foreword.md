---
title: 前言
---

Rust 编程语言在短短几年间走过了漫长的道路，从一个小而初生的爱好者社区创建和孵化，发展成为世界上最受欢迎和需求量最大的编程语言之一。回顾过去，Rust 的强大和前景必然会引起关注并在系统编程领域占据一席之地。然而，全球范围内对 Rust 的兴趣和创新增长渗透整个开源社区，并催化跨行业的广泛采用，这并非必然。

在此时刻，很容易指出 Rust 提供的出色功能来解释这种兴趣和采用的爆发。谁不想要内存安全、*并且*高性能、*并且*友好的编译器、*以及*出色的工具，还有众多其他出色功能？你今天看到的 Rust 语言结合了多年的系统编程研究与活跃而热情的社区的实践智慧。这门语言是有目的设计的，精心打造的，为开发者提供了一种工具，使他们更容易编写安全、快速和可靠的代码。

但让 Rust 真正特别的是它植根于赋能你——用户——实现你的目标。这是一门希望你成功的语言，赋能原则贯穿于构建、维护和推广这门语言的社区的核心理念。自这本权威著作的上一个版本以来，Rust 已进一步发展成为一种真正全球化和值得信赖的语言。Rust 项目现在得到 Rust 基金会的有力支持，该基金会还投资于关键举措，以确保 Rust 安全、稳定和可持续发展。

本版《Rust 程序设计语言》是全面的更新，反映了这门语言多年来的演变，并提供了有价值的新信息。但它不仅仅是语法和库的指南——它是加入一个重视质量、性能和深思熟虑设计的社区的邀请。无论你是希望首次探索 Rust 的经验丰富的开发者，还是希望精进技能的资深 Rustacean，本版都能为每个人提供所需的内容。

Rust 的旅程是协作、学习和迭代的旅程。这门语言及其生态系统的增长直接反映了其背后活跃而多元的社区。成千上万的开发者的贡献，从核心语言设计师到临时贡献者，使 Rust 成为如此独特而强大的工具。通过拿起这本书，你不仅仅是在学习一门新的编程语言——你是在加入一场让软件变得更好、更安全、更愉快使用的运动。

欢迎来到 Rust 社区！

- Bec Rumbul，Rust 基金会执行董事
