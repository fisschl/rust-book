# 介绍

> 注意：本书的这一版本与 [No Starch Press][nsp] 出版的[《Rust 程序设计语言》][nsprust]纸质版和电子书版本相同。

[nsprust]: https://nostarch.com/rust-programming-language-3rd-edition
[nsp]: https://nostarch.com/

欢迎阅读《Rust 程序设计语言》，一本关于 Rust 的入门书籍。Rust 编程语言帮助你编写更快、更可靠的软件。高级的人体工程学和低级的控制通常在编程语言设计中处于对立状态；Rust 挑战了这一冲突。通过平衡强大的技术能力和出色的开发者体验，Rust 让你能够控制低级细节（如内存使用），而无需传统上与这种控制相关的所有麻烦。

## Rust 适合谁

Rust 适合许多人，原因多种多样。让我们看看几个最重要的群体。

### 开发团队

Rust 被证明是在大型开发团队中协作的高效工具，这些团队的成员具有不同程度的系统编程知识。低级代码容易出现各种细微的 bug，而在大多数其他语言中，这些 bug 只能通过广泛的测试和经验丰富的开发者仔细的代码审查来发现。在 Rust 中，编译器扮演着守门人的角色，拒绝编译包含这些难以捉摸的 bug（包括并发 bug）的代码。通过与编译器一起工作，团队可以将时间花在关注程序的逻辑上，而不是追查 bug。

Rust 还将当代开发者工具带入系统编程世界：

- Cargo，内置的依赖管理器和构建工具，使添加、编译和管理依赖项在整个 Rust 生态系统中变得轻松而一致。
- `rustfmt` 格式化工具确保开发者之间的代码风格一致。
- Rust 语言服务器为集成开发环境（IDE）提供代码补全和内联错误信息的集成。

通过使用 Rust 生态系统中的这些和其他工具，开发者在编写系统级代码时可以提高生产力。

### 学生

Rust 适合学生和对学习系统概念感兴趣的人。使用 Rust，许多人学习了操作系统开发等主题。社区非常热情，乐于回答学生的问题。通过这本书等努力，Rust 团队希望让系统概念对更多人，特别是编程新手，更加易于理解。

### 公司

数百家大大小小的公司在生产中使用 Rust 执行各种任务，包括命令行工具、Web 服务、DevOps 工具、嵌入式设备、音视频分析和转码、加密货币、生物信息学、搜索引擎、物联网应用、机器学习，甚至 Firefox 网页浏览器的主要部分。

### 开源开发者

Rust 适合希望构建 Rust 编程语言、社区、开发者工具和库的人。我们很乐意让你为 Rust 语言做出贡献。

### 重视速度和稳定性的人

Rust 适合渴望在语言中追求速度和稳定性的人。说到速度，我们既指 Rust 代码运行的速度，也指 Rust 让你编写程序的速度。Rust 编译器的检查通过功能添加和重构确保稳定性。这与没有这些检查的语言中的脆弱遗留代码形成对比，开发者通常害怕修改这些代码。通过努力实现零成本抽象——编译成与手写代码一样快的高级功能——Rust 努力使安全代码也成为快速代码。

Rust 语言希望支持许多其他用户；这里提到的只是一些最大的利益相关者。总体而言，Rust 的最大抱负是通过提供安全*和*生产力、速度*和*人体工程学来消除程序员几十年来一直接受的权衡。试试 Rust，看看它的选择是否适合你。

## 本书适合谁

本书假设你已经在另一种编程语言中编写过代码，但不假设是哪一种。我们努力使内容能够被来自广泛编程背景的人广泛理解。我们不会花很多时间来讨论编程*是什么*或如何思考它。如果你完全是编程新手，你最好阅读一本专门提供编程入门介绍的书。

## 如何使用本书

一般来说，本书假设你按照从头到尾的顺序阅读。后面的章节建立在前面章节的概念之上，前面的章节可能不会深入讨论某个特定主题，但会在后面的章节中重新讨论该主题。

你会在本书中发现两种章节：概念章节和项目章节。在概念章节中，你将学习 Rust 的某个方面。在项目章节中，我们将一起构建小程序，应用你到目前为止学到的知识。第 2 章、第 12 章和第 21 章是项目章节；其余的是概念章节。

**第 1 章** 解释如何安装 Rust，如何编写 "Hello, world!" 程序，以及如何使用 Cargo，Rust 的包管理器和构建工具。**第 2 章** 是编写 Rust 程序的实践入门，让你构建一个猜数字游戏。在这里，我们在高层次上介绍概念，后面的章节将提供更多细节。如果你想立即动手，第 2 章是你的去处。如果你是一个特别细致的学习者，喜欢在进入下一个之前学习每个细节，你可能想跳过第 2 章，直接进入**第 3 章**，该章涵盖与其他编程语言相似的 Rust 特性；然后，当你想从事应用你所学到的细节的项目时，你可以回到第 2 章。

在**第 4 章**中，你将学习 Rust 的所有权系统。**第 5 章**讨论结构体和方法。**第 6 章**涵盖枚举、`match` 表达式以及 `if let` 和 `let...else` 控制流结构。你将使用结构体和枚举来创建自定义类型。

在**第 7 章**中，你将学习 Rust 的模块系统以及组织代码和其公共应用程序编程接口（API）的隐私规则。**第 8 章**讨论标准库提供的一些常见集合数据结构：向量、字符串和哈希映射。**第 9 章**探讨 Rust 的错误处理理念和技术。

**第 10 章**深入探讨泛型、trait 和生命周期，这些让你有能力定义适用于多种类型的代码。**第 11 章**全是关于测试的，即使有了 Rust 的安全保证，测试也是必要的，以确保你的程序逻辑正确。在**第 12 章**中，我们将构建我们自己的 `grep` 命令行工具功能子集的实现，该工具在文件中搜索文本。为此，我们将使用我们在前面章节中讨论的许多概念。

**第 13 章**探讨闭包和迭代器：Rust 中来自函数式编程语言的特性。在**第 14 章**中，我们将更深入地研究 Cargo，并讨论与他人共享你的库的最佳实践。**第 15 章**讨论标准库提供的智能指针以及启用其功能的 trait。

在**第 16 章**中，我们将介绍不同的并发编程模型，并讨论 Rust 如何帮助你无畏地进行多线程编程。在**第 17 章**中，我们通过探索 Rust 的 async 和 await 语法，以及任务、future 和流，以及它们启用的轻量级并发模型来进一步扩展。

**第 18 章**研究 Rust 惯用法与你可能熟悉的面向对象编程原则的比较。**第 19 章**是关于模式和模式匹配的参考，这是在 Rust 程序中表达想法的强大方式。**第 20 章**包含一系列高级主题，包括不安全 Rust、宏，以及更多关于生命周期、trait、类型、函数和闭包的内容。

在**第 21 章**中，我们将完成一个项目，在其中实现一个低级多线程 Web 服务器！

最后，一些附录以更参考性的格式包含有关该语言的有用信息。**附录 A** 涵盖 Rust 的关键字，**附录 B** 涵盖 Rust 的运算符和符号，**附录 C** 涵盖标准库提供的可派生 trait，**附录 D** 涵盖一些有用的开发工具，**附录 E** 解释 Rust 的版本。在**附录 F** 中，你可以找到本书的翻译版本，在**附录 G** 中，我们将介绍 Rust 是如何制作的以及什么是夜间版 Rust。

阅读本书没有错误的方式：如果你想跳到前面，去吧！如果感到困惑，你可能需要跳回前面的章节。但做任何适合你的事。
